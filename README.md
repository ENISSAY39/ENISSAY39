## 🚀 Projects

### 🤖 RAG Pipeline
Retrieval-Augmented Generation pipeline using Gemini, HuggingFace Sentence Transformers and Qdrant for semantic search, vector retrieval and AI-powered question answering.

### 🐾 Pet Catalog Manager
Full-stack web application developed with Laravel 12, PHP 8.4 and TailwindCSS featuring a complete CRUD system for pet management, advanced filtering, statistics dashboard with Chart.js, MVC architecture and relational database design.

### 📋 Task Manager
Collaborative full-stack task management platform built with Go, Gin, React and Dockerized MySQL including JWT authentication, role-based permissions, project/task management and modular frontend architecture.

### 🌐 E-Portfolio Platform
Personal e-portfolio web application developed with FastAPI, SQLModel and JWT authentication allowing users to manage professional experiences and education through a secure multi-user CRUD system.

### ☀️ SolPOC Scheduler
Planification et orchestration d’expériences de simulation avec système de cache, hashage et automatisation.
